# EnglishOCR
Synthesis Project in Artificial Intelligence course


### Objective
Building an end-to-end system for recognizing one handwriting english word using character segmentation and neural network. For more details, you can read our project's report [here](https://github.com/LekiizRaii/EnglishOCR/blob/main/assets/project_final_report.pdf) (it is in vietnamese)

### Tools
Python, OpenCV, TensorFlow, convolutional neural network, character segmentation.
